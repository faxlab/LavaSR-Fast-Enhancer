## What It Does

Got audio that sounds thin, muffled, or low-quality? This app uses an AI model called **LavaSR** to restore missing detail and bring your audio back to life ‚Äî then saves it as a clean **48 kHz WAV** file, ready for professional use.

It's designed to be fast and simple. Drop your files in, click **Enhance**, and you're done.

---

## Perfect For

- Voice recordings that sound narrow or dull
- Old interviews, podcasts, or phone call recordings
- Audio samples you want to polish before a mix
- Batch-cleaning a folder of files before editing

---

## Features

- **One-click enhancement** ‚Äî drop files, press Enhance, done
- **Batch processing** ‚Äî handle multiple files at once, even from different folders
- **Optional denoising** ‚Äî reduce background noise on top of enhancing
- **Drag output directly into your DAW or folder** ‚Äî no hunting through directories
- **Flexible output options** ‚Äî control the output folder, file suffix, and naming
- **Runs offline** ‚Äî no internet needed after the model downloads once

## Download & Install

### Option A ‚Äî Installer *(recommended)*

1. Go to [**Releases**](https://github.com/faxlab/LavaSR-Fast-Enhancer/releases/latest)
2. Download `LavaSR-Fast-Enhancer-Setup-*.exe`
3. Run it and follow the prompts
4. Open **LavaSR Fast Enhancer** from your Start Menu or desktop

### Option B ‚Äî Portable *(no install needed)*

1. Download `LavaSRFastEnhancer-Portable-*.zip` from [Releases](https://github.com/faxlab/LavaSR-Fast-Enhancer/releases/latest)
2. Extract the ZIP anywhere
3. Run `LavaSRFastEnhancer.exe`

> **First launch:** The AI model (~200 MB) will download automatically. This only happens once.

---

## How To Use

1. **Add your files** ‚Äî drag and drop audio files anywhere onto the app, or click **Select Files**
2. **Adjust settings** *(optional)* ‚Äî turn on denoising, change output folder, rename suffix
3. **Click Enhance** ‚Äî the app processes everything in the background
4. **Grab your results** ‚Äî drag finished files directly from the output list into your DAW, folder, or wherever you need them

### Supported Formats

`.wav` `.mp3` `.flac` `.ogg` `.m4a` `.aac` `.wma` `.aiff`

### Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl + O` | Select files |
| `Ctrl + Enter` | Enhance / Cancel |
| `Delete` | Remove selected files from queue |
| `Ctrl + L` | Clear the queue |
